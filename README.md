# mnist-computer-vision-model

1. This is a simple demonostrstion of the ability of models to identify hand written digits.

1. clone the repository in a new clean working space/ folder.

2.cd "mnsit-computer-vision-model"
3. run the ".py" script.
4. The model will automaticslly download datasets and train the model.
